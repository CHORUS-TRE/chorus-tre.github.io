# Open a Session

A session is your secure computing environment within a workspace. It is where you run applications, carry out analyses, or develop workflows.

To create a session:

* Enter the workspace you want to work in.
* Click Create Session.
* Provide the required information.
* Create the session, which becomes available for launch immediately.

Sessions can support multiple applications, allowing you to keep related tasks together in a controlled environment.
